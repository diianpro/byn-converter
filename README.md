# BYN Converter

Расширение для Chrome/Edge, конвертирует цены в белорусских рублях в USD/EUR/RUB
по официальному курсу НБ РБ. На kufar.by, realt.by и av.by работает из коробки.
Любой другой сайт включается одной кнопкой в popup'е («+ Включить на этом сайте»)
— расширение запросит permission именно на этот origin и зарегистрирует
content script через `chrome.scripting.registerContentScripts`.

Под капотом — единый универсальный детектор: `TreeWalker` идёт по text-нодам,
регулярка `\d+\s*(р\.?|BYN|руб\.?|Br)\b` ловит BYN-цену, рядом подставляется
суффикс `(~$X)`. Никаких сайт-специфичных селекторов.

## Установка

1. Распакуй архив.
2. Открой `chrome://extensions/` (или `edge://extensions/`).
3. Включи **Режим разработчика** (верхний правый угол).
4. Нажми **Загрузить распакованное расширение** и выбери папку `byn-converter`.
5. Зайди на kufar.by — рядом с ценами появятся конвертации: `1 250 р. (~$385)`.

## Структура

```
byn-converter/
├── manifest.json       # Manifest V3
├── background.js       # service worker: курсы НБ РБ + динамическая регистрация content scripts
├── content.js          # универсальный TreeWalker-детектор + MutationObserver для SPA
├── popup.html          # UI настроек
├── popup.js
└── icons/
```

## Как добавить новый сайт

Открыть сайт → открыть popup расширения → нажать **«+ Включить на этом сайте»**.
Chrome спросит permission на этот origin, расширение зарегистрирует content
script и на новой загрузке страницы заработает. Никаких правок кода.

## Источник курса

[НБ РБ](https://api.nbrb.by/exrates/rates?periodicity=0) — официальный курс,
обновляется раз в сутки в 00:00 минского времени. API бесплатный, без ключа и лимитов.

## Известные ограничения

- При нажатии «Включить на этом сайте» Chrome показывает системный prompt
  permission, и popup при этом закрывается. После grant'а открой popup
  заново — статус сайта обновится на «включён».
- Детектор ловит текст вида `число + р./BYN/Br/руб.`, поэтому редкие
  ложные срабатывания возможны (например, «купил за 100 р. в 1995-м»).
  Не критично.
- Используется официальный курс НБ РБ, реальный в обменниках на 1-3% отличается.
